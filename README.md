# x402 Payment Gateway

Production-ready x402 payment facilitator built with Convex + Next.js

## Packages

- **facilitator**: Core HTTP/402 handler, USDC verification, fee calculation
- **core**: SDK types, utilities, constants
- **agent**: Agent integration helpers
- **cli**: Command-line tools
- **react**: React components for x402 payment flows
- **nextjs**: Next.js integration
- **openapi**: API schema and docs

## Quick Start

\`\`\`bash
pnpm install
pnpm dev
\`\`\`

## Deployment

- Facilitator runs on any Node.js host (Railway, Render, Vercel)
- Convex backend (serverless, real-time subscriptions)
- Base network (USDC mainnet)
